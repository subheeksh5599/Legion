# Legion

Autonomous adversarial agents that probe legacy systems for vulnerabilities before attackers do.

## Stack

- **Frontend:** Next.js 16, React 19, Tailwind CSS 4, motion
- **Agent Framework:** Fetch.ai uAgents
- **Scoring:** Bittensor Subnet
- **Privacy:** Venice.ai TEE
- **Audit Trail:** Solana
- **Governance:** Conduct AI

## Run

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # Production build
```
